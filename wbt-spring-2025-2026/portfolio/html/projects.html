  2. Graphics design using OpenGL with C++

    3. Project name: Animated City Road Scenario

    Detail: The “Animated City Road Scenario” is an interactive OpenGL (C++) project that simulates a lively city environment with a two-lane road, moving vehicles, zebra crossing, buildings, trees, field, lake, and animated sky elements like the sun, clouds, birds, and airplane. Users can interact through a button-controlled pedestrian crossing and toggle between day and night modes with dynamic lighting changes. The project demonstrates graphics programming concepts including animation, object rendering, transformations, and real-time input handling.
